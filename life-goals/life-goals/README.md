# life-goals

A living goal system for building an intentional life across all domains — career, family, financial security, health, personal growth, and legacy.

Synthesizes GTD Horizons, OKRs, Atomic Habits (James Clear), and the 12 Week Year into a continuous loop: **define → plan → execute → review → redefine**.

## Install

```
/plugin install life-goals@mab-agent-skills
```

## Skills

### `life-goals`

Triggered by phrases like "help me with my goals", "let's review my goals", "I need to plan for the future", "what should I be working on", "help me set priorities", or "I want to be more intentional about my life".

**Modes:**

| Mode | Description |
|------|-------------|
| **Setup** | First-time or full-reset. Guides through north star discovery, time horizon cascade (30yr → 10yr → 3yr → 1yr → 90-day), and Atomic Habits-based habit design. Produces a persistent `goals/GOALS.md`. |
| **Review** | Weekly (15 min), monthly (30 min), quarterly (60-90 min), or annual (2-3 hr) check-ins with structured templates. |
| **Regenerate** | For when life changes significantly — updates affected domains without discarding the full system. |
| **Quick Check** | Fast orientation: what's the active sprint, what matters this week, which habits need attention. |

**Persistent state:** `goals/GOALS.md` in your workspace — one document the skill reads and updates across sessions.

**Reference files bundled:**
- `frameworks.md` — GTD, OKRs, Atomic Habits, 12 Week Year, retirement math
- `domains.md` — Deep-dive on each life domain with guiding questions
- `review-templates.md` — Structured templates for each review cadence
